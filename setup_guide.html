Local Environment Setup

Workspace & Supabase Blueprint

An interactive step-by-step developer walkthrough to create a Supabase database, configure environment variables, and install dependencies locally.

1 Create & Provision Your Supabase Project

Supabase acts as your secure backend database. To set it up and obtain your custom API keys, follow these instructions:

  1. Go to the Supabase Dashboard ↗ and sign in using your GitHub account.
  2. Click on the + New Project button.
  3. Select or create an organization, then fill in the project settings:
    • Name: cy-global-merchants
    • Database Password: Create a strong password (save it safely).
    • Region: Select the region closest to your main user base.
    • Pricing: Choose the Free tier.
  4. Click Create New Project. The provisioning process will take approximately 1-2 minutes to prepare your secure database instance.

💡 How to find your API credentials:
Once provisioning completes, navigate to Project Settings (gear icon at bottom left) ➔ API. Here you will find your custom Project URL and your anon public API key.

2 Create Local Environment File

Next.js is designed to load your secret variables securely from local files. In the root directory of your workspace, create a file named .env.local and paste your custom credentials inside it.

# Supabase Credentials (Pre-configured for project faywvbptdyfydzgocenk)
NEXT_PUBLIC_SUPABASE_URL=https://faywvbptdyfydzgocenk.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-public-key-here

# Resend Email Integration API Key
RESEND_API_KEY=re_your_secret_resend_api_key

# Analytics & Tracking (Optional)
NEXT_PUBLIC_GA_ID=
NEXT_PUBLIC_CLARITY_ID=

⚠️ Important Security Note: The .env.local file is already ignored in your .gitignore, ensuring your private database passwords and secret API keys are never leaked to public repositories on GitHub.

3 Install Project Dependencies

Now, set up your development environment locally. Open your terminal in the project directory (cy-global-merchants) and execute the following commands:

  1. Make sure Node.js (LTS version) ↗ is installed on your local computer.
  2. Install all required dependencies (React 19, Next.js 15, Framer Motion, TailwindCSS, etc.) by running:
npm install

This command downloads and configures all dependencies declared inside your package.json file into a local node_modules directory.

4 Launch the Development Server

Once installation finishes and your environment keys are in place, spin up the local server to run your site:

npm run dev

Your local server will boot up and host the live development site at:

👉 http://localhost:3000 ↗
